Speech Recognition Commands

Command Description
scroll up Scrolls the page upward
scroll down Scrolls the page downward
go to top Goes to the top of the page
go to bottom Goes to the bottom of the page
go back Goes back to the previous page
go forward Goes forward to the next page
open new tab Opens a new browser tab
open website Opens a specific website
close tab Closes the current browser tab
reload page Reloads the current page
stop loading Stops the page from loading
zoom in Zooms in on the page
zoom out Zooms out on the page
zoom reset Resets the zoom level to 100%
search Performs a search using a search engine
select all Selects all the text on the page
copy Copies the selected text
cut Cuts the selected text
paste Pastes the copied or cut text
undo Undoes the last action
redo Redoes the previously undone action
